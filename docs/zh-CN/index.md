## 语言说明

- 当前仓库的 GitHub / Markdown 文档采用**静态多语言文档树**方案，而不是运行时自动翻译。
- **英语**与**简体中文**目前提供可连续导航的公开入口链路。
- 其它语言当前仍主要是入口页或白皮书入口，待对应文档树补齐后再升级为完整切换。

## 公开边界

- 英文白皮书 [`../WHITEPAPER.md`](../WHITEPAPER.md) 仍然是当前公开规范性源文本。
- 中文文档的目标是提供**同语言连续阅读体验**，不是创建与英文原文相冲突的独立规范体系。
- 若不同语言版本之间出现措辞或数值歧义，以英文规范性文本和最终已审计、已发布的链上 / 合约资料为准。
