# ION — Операционная система для сверхцифровой цивилизации

Это русскоязычная входная страница основного GitHub README проекта ION.

ION — не обычный DEX-репозиторий, а долгосрочная цифровая система, которая начинается с инфраструктуры обмена и постепенно расширяется в платежи, идентичность, доказуемость, репутацию, координацию и самоэволюцию.
