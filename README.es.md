## Qué es ION

ION no es un proyecto DEX común.
Se define como un sistema digital de largo horizonte que comienza con infraestructura de intercambio y crece hacia pagos, identidad, prueba, reputación, arbitraje, coordinación y autoevolución.

El README principal en inglés sigue siendo la versión más completa del frente global del proyecto.
