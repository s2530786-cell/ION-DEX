# 商户接入

> 面向商户与电商接入方的中文公开入口。

## 为什么商户会关心 ION DEX

| 项目 | 传统处理方 | ION DEX 目标方向 |
|------|------------|------------------|
| 结算时间 | 2–7 天 | 秒级到十几秒级 |
| 交易费率 | 2%–3% | 公开草案中约 `0.5%` |
| 跨境摩擦 | 多处理方与换汇成本 | 统一 ION 轨道 |
| Web3 用户触达 | 无 | 可直接接入 |
| 链复杂度 | 不适用 | 由基础设施吸收 |

## 常见接入路径

### 1. 支付按钮

适合最轻量的 checkout 集成。

### 2. API 集成

适合自定义订单流、支付会话与 webhook 回调。

### 3. 直接结算

适合高体量商户直接进入 ION 结算逻辑。

## 支持币种方向

### 用户前端可能支付的资产

- USDT
- USDC
- ION
- 其它由 ION DEX 流动性池支持的主流资产

### 商户后端接收方向

- 默认以 **ION** 结算
- 部分场景可设计为自动转换到稳定币

## 手续费说明

当前公开文档中商户相关费率应被视为**公开草案**，不是最终主网常量：

- 平台费常见草案：`0.5%`
- 其中销毁 / 质押 / 国库分配仍待最终已审计合约与发布文档确认

## 接入流程建议

1. 联系项目方登记商户身份
2. 获取测试环境或目标 API 凭证
3. 配置 webhook 回调
4. 在测试环境完成支付闭环验证
5. 确认对账与浏览器验证链路
6. 再进入正式上线准备
