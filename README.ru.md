## Что такое ION

ION — это не обычный DEX-проект.
Он рассматривается как долгосрочная цифровая система, которая начинается с инфраструктуры обмена и постепенно расширяется в сторону платежей, идентичности, доказуемости, репутации, арбитража, координации и самоэволюции.

Основной английский README пока остаётся самой полной глобальной входной версией проекта.
Эта страница будет постепенно расширяться как русскоязычное зеркало.
