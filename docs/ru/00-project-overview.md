# Project Overview

ION DEX is planned as a full-stack Web3 trading and ecosystem portal for Ice Open Network. It combines professional spot trading, AMM liquidity, staking, burn analytics, bridge flows, ION DNS domains, ION Identity, AI market intelligence, and transparent treasury operations.
