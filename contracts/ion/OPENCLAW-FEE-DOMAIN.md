# OpenClaw 手续费模块保留区

## 禁止 AI Agent 写入/修改

本目录及以下任何 `.fc` 文件**禁止任何 AI Agent（Cursor/Codex/等）写入或修改**。

```
contracts/ion/*.fc
```

## 维护方

由 **OpenClaw + 豆包 API** 手动维护。

## 手续费费率（全局固定）

总 Swap 手续费：**0.3%**，内部分配比例：

| 接收方 | 比例 | 说明 |
|--------|------|------|
| LP 流动性提供者 | 50% | 按份额分配 |
| ION 质押者 | 15% | 按质押量分配 |
| 国库 | 10% | 项目开发基金 |
| 开发者 | 25% | 硬编码开发者钱包提取 |

## 约束

- `contracts/ion/ion_pool.fc` 仅将协议费**全额转发**至部署时配置的 `fee_sink` 地址
- fee_sink 地址不实现权重分润逻辑
- 分润逻辑由 OpenClaw 维护的独立模块处理
- 开发者钱包提现权限**硬编码在合约中**
