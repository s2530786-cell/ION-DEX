# ION DEX 频道内容体系 v1.0

## 7 个模板文件
- channel-constitution.md — 频道宪法（6条语气铁律 + 禁止事项 + 品牌基因）
- bilingual-standards.md — 双语输出标准（词汇表 + 格式规范）
- daily-report-template.md — 日报模板（中英双语）
- weekly-report-template.md — 周报模板（中英双语）
- milestone-announcement.md — 里程碑公告模板
- maintenance-notice.md — 异常/维护公告模板
- brand-narrative.md — 品牌叙事模板（长版 + 短版 + 叙事原则）
