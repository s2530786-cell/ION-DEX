## 双语输出标准

### 核心原则

1. **中文原生** — 中文版是第一稿，不是翻译
2. **英文专业** — 英文版是专业翻译，不是机翻
3. **分段不混排** — 中文段 + 英文段，中间空行分隔

### 示例格式

```
中：ION DEX 交换功能已完成编译验证，进入安全测试阶段。
EN: ION DEX swap module has passed compilation and entered the security testing phase.

中：今日新增 3 项修复，涵盖重入保护与权限校验。
EN: 3 fixes added today, covering reentrancy guards and access control.
```

### 词汇表

| 中文 | English |
|------|---------|
| 交换 | Swap |
| 流动性池 | Liquidity Pool |
| 质押 | Staking |
| 跨链桥 | Bridge |
| 手续费 | Protocol Fee |
| 销毁 | Burn |
| 安全测试 | Security Audit |
| 部署 | Deployment |
| 主网 | Mainnet |
| 测试网 | Testnet |
| 编译 | Compilation |
| 修复 | Fix / Patch |
| 新增功能 | Feature |
| 已上线 | Live |

### 禁止

- ❌ 中英混杂一行（"Swap 功能已 fix"）
- ❌ 机翻英文（语法不通 / 用词诡异）
- ❌ 纯中文发帖不含英文

### 工具链

- 英文草稿用 AI 辅助润色，但人工检查术语准确性
- 最终英文版应达到母语者可读水平
