# 路线图指南

> 从品牌基础到成熟数字文明的阶段化执行导读。

## 当前定位

**当前公开仓库可见部分大致处于 P0–P1。**

- P0：品牌与论纲已经成形
- P1：DEX 基础设施处于持续工程化中
- P2 / P3：身份与支付层仍以设计 / 集成工作为主
- P4 以后：主要属于路线图规划

## 阶段概览

### P0：品牌与论纲

目标：建立公开定位、旗舰 README、白皮书、官方渠道与文明级叙事。

### P1：DEX 基础设施

目标：逐步构建 28 链去中心化交易能力，包括：

- swap 路由
- 流动性池
- 网格交易
- 限价单
- 手续费收集与分发

### P2：身份与证明

目标：构建 ION Identity，形成跨生态连续身份、不可变历史与可验证 standing。

### P3：支付轨道

目标：构建低摩擦支付基础设施，使 ION 能支撑更广的商业结算场景。

### P4：AI 仲裁与防护

目标：引入 AI 辅助纠纷处理、Sentinel 风险监控与更强的安全治理工具。

### P5：自我进化

目标：把学习、纠错、规则修正与连续性适配内建进平台结构。

### P6：商户与电商

目标：为商户和电商提供更强的收款、结算与集成轨道。

### P7：现实服务

目标：把 ION 轨道扩展到配送、出行、保险与物流。

### P8：经济集成

目标：激活销毁、质押、国库与动态费率飞轮。

### P9：治理与反滥用

目标：在保持用户主权的同时建立更成熟的治理与反滥用体系。

### P10：成熟文明

目标：形成一个面向全球、可持续扩张、能自我治理和自我修正的数字文明栈。

## 阅读提醒

- 路线图描述的是**目标阶段图**，不等于每一阶段都已在当前仓库中实现。
- 当你看到“planned / in design / roadmap”相关内容时，应与“当前可验证能力”区分阅读。
