# ION — 초디지털 문명을 위한 운영체제

ION GitHub 메인 페이지로 연결되는 한국어 페이지입니다.

ION은 일반적인 DEX 저장소가 아닙니다. 거래 인프라를 출발점으로 삼아 결제, 신원, 증명, 평판, 조정, 자기 진화로 확장되는 장기 지향 디지털 시스템입니다.
