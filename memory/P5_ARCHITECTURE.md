## 📊 价格计算流程

```
输入: 100 ION
  │
  ├─ 1️⃣ 获取交易对价格
  │   └─ ION/USDT = 4.82
  │
  ├─ 2️⃣ 计算基础输出
  │   └─ baseOutput = 100 × 4.82 = 482 USDT
  │
  ├─ 3️⃣ 计算价格影响
  │   ├─ 流动性池: 12,500,000 USDT (Mock)
  │   ├─ 影响率: (100 / 12,500,000) × 100 = 0.0008%
  │   └─ 限制: min(0.0008%, 15%) = 0.0008%
  │
  ├─ 4️⃣ 计算滑点
  │   ├─ 模式检查:
  │   │   Auto → suggestedSlippage (0.5%)
  │   │   Custom → customSlippage (用户输入)
  │   └─ 实际滑点: 0.5%
  │
  ├─ 5️⃣ 计算最小输出
  │   └─ minOutput = 482 × (1 - 0.5%) = 479.59 USDT
  │
  ├─ 6️⃣ 计算费用
  │   ├─ 交易费: 482 × 0.3% = 1.446 USDT
  │   └─ ION费: 100 × 1% = 1.0 ION
  │
  ├─ 7️⃣ 选择路由
  │   └─ ION → USDT (直接)
  │
  └─ 8️⃣ 返回完整报价
      {
        baseOutput: 482,
        slippage: 0.5,
        minOutput: 479.59,
        priceImpact: '0.08',
        tradingFee: 1.446,
        ecoFeeInION: 1.0,
        route: ['ION', 'USDT']
      }
```

## 🎯 滑点管理流程

```
用户操作
  │
  ├─ 选择 [Auto 模式]
  │   │
  │   ├─ 检查价格影响
  │   │   ├─ < 0.1%   → 推荐 0.5%
  │   │   ├─ < 1%     → 推荐 1.0%
  │   │   ├─ < 5%     → 推荐 2.0%
  │   │   ├─ < 10%    → 推荐 5.0%
  │   │   └─ ≥ 10%    → 推荐 10.0%
  │   │
  │   └─ suggestedSlippage = 自动计算值
  │
  ├─ 选择 [Custom 模式]
  │   │
  │   ├─ 点击预设按钮 (0.5% / 1.0% / 2.0% / 5.0%)
  │   │   或手动输入
  │   │
  │   ├─ 验证输入范围
  │   │   ├─ < 0.1%  ❌ 太低，交易可能失败
  │   │   ├─ 0.1-50% ✅ 合理范围
  │   │   └─ > 50%   ❌ 太高，损失过大
  │   │
  │   └─ customSlippage = 用户输入值
  │
  └─ 显示风险警告
      ├─ 低影响 (< 2%) → 🟢 绿色：安全
      ├─ 中影响 (2-5%) → 🟡 橙色：需注意
      └─ 高影响 (> 5%) → 🔴 红色：需谨慎
```

## 🧮 关键计算公式

### 1. 基础输出
```
baseOutput = inputAmount × exchangeRate
例: 100 ION × 4.82 = 482 USDT
```

### 2. 价格影响
```
priceImpact = (inputAmount / liquidity) × 100
例: (100 / 12,500,000) × 100 = 0.0008%
上限: min(calculated, 15%)
```

### 3. 最小输出
```
minOutput = baseOutput × (1 - slippage/100)
例: 482 × (1 - 0.5/100) = 479.59 USDT
```

### 4. 交易费
```
tradingFee = baseOutput × 0.3%
例: 482 × 0.003 = 1.446 USDT
```

### 5. 生态费
```
ecoFeeInION = inputAmount × 1%
例: 100 × 0.01 = 1.0 ION
```

### 6. 动态滑点建议
```
IF priceImpact < 0.1% THEN suggestedSlippage = 0.5%
IF priceImpact < 1%   THEN suggestedSlippage = 1.0%
IF priceImpact < 5%   THEN suggestedSlippage = 2.0%
IF priceImpact < 10%  THEN suggestedSlippage = 5.0%
ELSE                  THEN suggestedSlippage = 10.0%
```
