## ION DEX 简介

- 以 28 链 DEX 与聚合交易入口为起点的长期基础设施。
- 面向用户、商户与平台的低摩擦多链支付能力。
- 统一的身份、证明、信誉与可验证历史层。
- 围绕 AI Sentinel、防御、仲裁与协同的长期架构。
- 以 burn、staking 与 treasury 为核心的长期价值飞轮。
